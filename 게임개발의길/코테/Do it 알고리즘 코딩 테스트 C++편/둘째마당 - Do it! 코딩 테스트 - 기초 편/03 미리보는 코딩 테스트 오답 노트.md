# 03-4 나머지 연산의 중요성 알아보기

자료형의 표현 범위를 넘지 않으면서 나머지 연산을 할 때는 나머지 연산의 분배 법칙을 이용한다.
덧셈, 뺄셈, 곱셈의 분배 법칙이 성립한다.
![[Pasted image 20251212141746.png]]
예를 들어 1부터 50까지 곱한 값을 10007로 나눈 나머지를 구하는 과정을 진행한다고 가정했을 때,
1부터 50까지 곱한 값을 먼저 구한 다음 그걸 10007로 나눈다고 생각해 본다.
그럼 곱셈 과정에서 이미 long long의 표현 범위를 초과해 버린다.
여기서 분배 법칙을 활용하면, 각각의 수에서 10007을 나눈 나머지를 구하는 연산을 각 반복마다 진행할 수 있다. 
이런 식으로 나머지 연산 시 응용이 가능하다.
# 03-5 정렬 기초 다지기

정렬은 코딩 테스트에서 필수적인 요소이다. 정렬의 기본 사용법을 익혀보자.
먼저 오름차순 정렬은 sort() 함수로 진행한다.
내림차순 정렬도 마찬가지로 sort()로 진행하는데, 3번째 파라미터에
내림차순 비교 함수 객체인 greater\<int\>() 를 적용한다.
# 03-6 다중 조건 정렬 익히기

C++에서는 다중 조건 정렬을 2가지 방법으로 구현할 수 있다.
첫째는 sort()에 사용자 정의 비교 함수를 입력하는 것이고,
둘째는 구조체 내부에서 비교 연산자 오버로딩을 통해 진행한다.
