## 직렬화란
오브젝트나 연결된 오브젝트의 묶음을 바이트 스트림으로 변환하는 과정

**직렬화의 장점**
현재 상태를 저장하고 필요할 때 복원 가능
네트워크를 통해 현재 프로그램의 상태를 다른 컴퓨터에 복원 가능
데이터 압축, 암호화를 통해 데이터를 효율적이고 안전하게 보관 가능
## 언리얼 엔진의 직렬화 시스템
FArchive 클래스를 제공
<< operator 연산자 사용
다양한 아카이브 클래스 제공
json 직렬화 기능 별도 라이브러리 통해 제공
## json 직렬화
json 장점: 데이터 크기가 가볍고 읽기 편함. 웹 통신 표준으로 널리 사용됨
단점: 지원 타입이 몇가지 안되고 텍스트 형식으로만 사용 가능
## 언리얼 스마트 포인터 라이브러리
TUniquePtr: 지정한 곳에서만 메모리를 관리하는 포인터
TSharedPtr: 더 이상 사용되지 않으면 자동으로 메모리를 해지하는 포인터
TSharedRef: 공유포인터와 동일하지만 유효한 객체를 항상 보장받는 레퍼런스
