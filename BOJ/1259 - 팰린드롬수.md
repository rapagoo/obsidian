https://www.acmicpc.net/problem/1259

```cpp
#include <iostream>
#include <string>
#include <algorithm>

int main()
{
	while (1)
	{
		std::string str{};
		std::cin >> str;
		if (str == "0") break;
		std::string copied = str;
		std::reverse(str.begin(), str.end());
		if (str == copied) std::cout << "yes" << '\n';
		else std::cout << "no" << '\n';
	}
}
```

std::reverse 를 이용해서 문자열을 뒤집은 다음 원본과 비교한다.
