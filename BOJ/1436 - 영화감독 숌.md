https://www.acmicpc.net/problem/1436

```cpp
#include <iostream>
#include <string>
using namespace std;
int main()
{
    int num;
    cin >> num;
    int N{665};
    int count{};
    while (true)
    {
        string s{to_string(N)};
        if (s.find("666") != string::npos)
            ++count;
        if (count == num)
        {
            cout << N << endl;
            break;
        }
        ++N;
    }
    return 0;
}
```

숫자를 문자열로 바꾸고, 문자열 내에서 특정 문자열을 검색하는 것이 이 문제의 핵심이다.
문자열 검색 후에는 npos 와 비교하는 것이 기본이다. 
npos 는 위치 없음(no position) 의 약자이다.
find 함수와 보통 묶어서 사용한다.
find 함수의 return value는 찾은 부분의 문자열이 시작되는 위치이다.
