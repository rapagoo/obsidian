https://www.acmicpc.net/problem/11650

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <utility>
int main()
{
    int N;
    std::cin >> N;
    std::vector<std::pair<int, int>> coords(N);
    for (int i{}; i < N; ++i)
    {
        int a, b;
        std::cin >> coords[i].first >> coords[i].second;
    }
    std::sort(coords.begin(), coords.end());
    for (const auto &p : coords)
    {
        std::cout << p.first << " " << p.second << "\n";
    }
    return 0;
}
```

std::pair, std::vector, std::sort 이 세가지 개념을 주목하자.
또한 range based for loop를 사용한 것도 기억하자.
2차원 좌표를 배열에 저장해야 하는데, 배열의 크기도 사용자가 입력하는 숫자에 맞춰 정해야 하기 때문에 vector를 사용한다. 배열의 타입은 pair를 사용해 int 두개를 한 쌍으로 받는다.
sort 알고리즘은 오름차순으로 정렬해주는 알고리즘이다. std::pair를 정렬할 경우, 별도 설정 없이 `first` 기준으로 먼저 정렬하고, `first`가 같으면 `second` 기준으로 자동 정렬한다.
for loop에서 const 와 & 를 사용한 것도 기억하자. const를 통해 원본 배열을 수정할 수 없도록 하고, &로 불필요한 복사가 일어나지 않도록 한다. 읽기만 할 시에 가장 효율적인 방식이다.
