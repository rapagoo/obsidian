https://www.acmicpc.net/problem/1924

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	int x, y;
	cin >> x >> y;
	int month[12] = { 31,28,31,30,31,30,31,31,30,31,30,31 };
	string date[7] = { "SUN", "MON", "TUE", "WED", "THU", "FRI", "SAT" };
	int totaldays{ 0 };
	totaldays += y;
	for (int i{ 0 }; i < (x - 1); ++i) {
		totaldays += month[i];
	}
	cout << date[totaldays % 7];
	return 0;
}
```

오늘이 무슨 요일인지 계산하는 문제이다.
월별로 몇일까지 있는지 배열에 순서대로 저장하고, 만약 현재 3월의 특정 요일을 계산한다면, 1월과 2월의 전체 일수를 더한 값에 3월 x일의 x를 더해, 7로 나눈 값의 나머지에 따라 요일을 출력한다.
