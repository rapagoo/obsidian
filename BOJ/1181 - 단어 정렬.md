https://www.acmicpc.net/problem/1181

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

bool compare(const std::string& a, const std::string& b)
{
	if (a.length() == b.length())
		return a < b;
	return a.length() < b.length();
}

int main()
{
	int N{};
	std::cin >> N;
	std::vector<std::string> v(N);
	for (int i{}; i < N; ++i)
	{
		std::cin >> v[i];
	}
	std::sort(v.begin(), v.end(), compare);
	v.erase(std::unique(v.begin(), v.end()), v.end());
	for (std::string s : v)
	{
		std::cout << s << '\n';
	}
	return 0;
}
```

문자열들을 여러 개 저장하고, 정렬하는 문제이다.
compare 함수에서 return 값을 어떻게 지정하는지가 중요하다.
두 문자열의 길이가 같다면, 사전 순으로 나열한다. 
`return a < b;` 하게 되면, a가 b보다 사전순으로 앞이라면 true를 반환한다.
두 문자열 길이가 다르다면, a가 b보다 짧을 때 true를 반환한다.
이를 sort 함수에 비교 함수로 사용한다.
또한 unique 를 사용해 vector의 중복된 값을 다 뒤로 밀고, 중복된 값들이 시작되는 지점을 반환한다.
이 반환점부터, vector의 끝까지 erase를 사용해 지운다.
