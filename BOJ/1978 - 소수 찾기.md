https://www.acmicpc.net/problem/1978

```cpp
#include <iostream>

bool isPrime(int n)
{
	if (n < 2) return false;
	for (int i{ 2 }; i * i <= n; ++i)
	{
		if (n % i == 0) return false;
	}
	return true;
}

int main()
{
	int N{};
	std::cin >> N;
	int cnt{};
	for (int i{}; i < N; ++i)
	{
		int n{};
		std::cin >> n;
		if (isPrime(n)) ++cnt;
	}
	std::cout << cnt;
	return 0;
}
```

소수 판정법만 알면 되는 문제다. 소수 판정법을 기억하자.
