https://www.acmicpc.net/problem/11719

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	string str;
	while (getline(cin, str)) {
		cout << str << endl;
	}
	return 0;
}
```

한 줄의 내용을 그대로 읽어오는 `getline` 함수를 알고 있어야 한다.
`getline` 을 while 안에 넣어서, 입력 스트림에서 읽을 내용이 없다면 루프를 종료시킨다
