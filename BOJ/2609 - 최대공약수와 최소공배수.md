https://www.acmicpc.net/problem/2609

```cpp
#include <iostream>
using namespace std;
int main()
{
    int a, b;
    int GCD, LCM;
    cin >> a >> b;
    LCM = a * b;
    while (b != 0) { int r = a % b; a = b; b = r; }
    GCD = a;
    LCM /= GCD;
    cout << GCD << endl << LCM << endl;
    return 0;
}
```

`while (b != 0) { int r = a % b; a = b; b = r; }` 이 문장이 핵심이다.
유클리드 호제법을 코드로 구현한 것. 최대공약수를 구할 때 주로 사용하는 방식이라고 하니 꼭 알아두자.
최소공배수는 최대공약수를 구하기만 하면 아주 쉽게 구할 수 있다. 두 수를 곱한 값을 최대공약수로 나누면 된다.
